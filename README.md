# CodeInit.dev

Nesse tutorial vou mostrar como criar sua primeira API usando Node e Express.

## Instruções:

1. Clone o projeto.
2. Instale as dependências. Estou usando o Yarn, mas você pode optar pelo gerenciador de sua preferência:

```sh
yarn install
```

3. Execute o servidor
```sh
yarn dev
```
